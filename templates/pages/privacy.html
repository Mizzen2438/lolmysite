プライバシーポリシー

最終更新日: 2026年6月20日

NEONQ(以下「本サービス」)は、利用者の個人情報を以下の方針に基づいて取り扱います。運営者は NEONQ運営 です。

1. 取得する情報

  • Discord アカウント情報: Discord OAuth により取得する Discord ユーザー ID・ユーザー名・アバター画像 URL・アカウント作成日時。
  • Riot Games アカウント情報: 利用者が登録した Riot ID(ゲーム名 + タグライン)、および Riot Games API から取得する PUUID・ランク(ソロ/フレックス)。
  • プロフィール情報: 得意レーン、プレイ時間帯、ボイスチャットの希望、自己紹介。
  • 利用情報: 募集・応募・通報・ブロック等の操作履歴、アクセスログ。

2. 利用目的

  • ゲーム仲間のマッチング(募集・応募・参加者管理)機能の提供
  • ランクに基づく募集条件の判定・表示
  • 不正行為の防止、通報対応、コミュニティの健全性維持
  • サービスの改善および障害対応

3. 第三者サービスへの提供・連携

本サービスは認証およびゲーム情報取得のため、以下の外部サービスと連携します。各社のプライバシーポリシーが適用されます。

  • Discord(認証): Discord プライバシーポリシー
  • Riot Games(ゲームアカウント・ランク取得): Riot Games プライバシー通知

これらの目的以外で、利用者の同意なく個人情報を第三者へ提供することはありません(法令に基づく場合を除く)。

4. 保存期間と削除

個人情報はアカウントが有効な間保持します。利用者は退会により個人情報の削除を求めることができます。退会時、個人情報は削除され、募集・応募履歴は匿名化して保持する場合があります。なお、不正対策のため、凍結等の措置に関する記録(Discord ID 単位)は退会後も保持することがあります。

5. データの保護

通信は HTTPS で暗号化し、認証トークンや API キーはサーバー側で安全に管理します。

6. お問い合わせ

個人情報の開示・訂正・削除のご請求やお問い合わせは contact@neonq.online までご連絡ください。

7. 改定

本ポリシーは必要に応じて改定し、本ページに掲示します。
